# 사업장및사회보험관리

이 리포지터리는 `proto.html` 파일을 포함합니다. 이 파일을 GitHub에 푸시하고 GitHub Pages로 퍼블리시하면 URL로 공유할 수 있습니다.

퍼블리시 방법:
- GitHub 웹 UI에서 리포지터리를 생성한 뒤 로컬 리포지터리를 푸시하고 `Settings → Pages`에서 브랜치를 선택해 퍼블리시
- 또는 `gh` CLI가 설치되어 있고 로그인 되어 있다면 아래 명령으로 자동 생성 및 푸시 가능:

```bash
# 예시: gh repo create <REPO_NAME> --public --source=. --remote=origin --push
```
